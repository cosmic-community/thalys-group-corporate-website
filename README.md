## Getting Started

### Prerequisites

- Node.js 18+ or Bun
- Modern web browser

### Installation

1. Clone the repository
```bash
git clone <repository-url>
cd thalys-group-website
```

2. Install dependencies
```bash
bun install
```

3. Start the development server
```bash
bun dev
```

4. Open [http://localhost:3000](http://localhost:3000) in your browser
